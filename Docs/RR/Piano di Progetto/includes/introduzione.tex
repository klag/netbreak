\subsection{Scopo del prodotto}
Lo scopo del prodotto è la realizzazione di un API Market per l'acquisto e la vendita di microservizi. Il sistema offrirà la possibilità di registrare nuove API per la vendita, permetterà la consultazione e la ricerca ai potenziali acquirenti, gestendo i permessi di accesso/utilizzo tramite creazione e controllo di relative API key. Il sistema, oltre alla web app stessa, sarà corredato di un API Gateway\ped{\textit{G}} per la gestione delle richieste e il controllo delle chiavi, e fornirà funzionalità avanzate di statistiche per il gestore della piattaforma e per i fornitori dei microservizi.
