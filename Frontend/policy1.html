Policy per numero di chiamate

Si paga solo in base al numero di chiamate effettuate. Questa policy e' stata pensata per API con traffico e tempo di utilizzo costanti. Questa policy e' stata pensata per API con traffico e tempo di utilizzo costanti. La vendita avverra' prima dell'uso, a blocchi di 1*10x o 2*10x o 5*10x, con x ∈ [2,3,4], resi disponibili a seconda della tipologia di utente e l'utilizzo potra' essere monitorato in tempo reale, con la possibilita' da parte di un utente di ricevere un'email raggiunto un opportuno ε di chiamate rimanenti.

CM Costo mantenimento API Market per chiamata. Serve per evitare che API Market vada in perdita
CS Guadagno impostato dallo sviluppatore
%APIMarket Percentuale trattenuta da API Market sul guadagno impostato
NC Numero di chiamate che il cliente vuole acquistare

formula: (CM + GS + %APIMarket) * NC
